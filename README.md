# Chefdist

Chefdist creates a distributable chef repo that can be used with chef-solo / chef-zero (Serverless Chef)

## Installation

Add this line to your application's Gemfile:

    gem 'chefdist'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install chefdist
